# 刷题 App 5.0 使用帮助

> 适用版本：5.0.0
> 更新日期：2026-07-13

## 1. 登录与首页

使用分配的工号或邮箱和密码登录。登录后首页会显示可用题库、总题数、练习次数和每个题库的学习状态。

- “进入”打开题库详情。
- “刷题”直接开始该题库的全部题型。
- “缓存”把云端题库保存到当前设备，便于离线练习。
- “上传题库”导入自己的题库。
- “错题练习”进入当前账号的需复习题目。

管理员账号会额外显示“管理”栏目。

## 2. 导入与上传题库

支持 DOCX、TXT、JSON、CSV 和 Markdown。

DOCX 会尝试识别单选、多选、填空、判断、简答、背记题和题目配图。导入后先检查预览中的题型、题干、答案和图片，再确认保存。

本地保存只在当前浏览器可见。上传云端后：

1. 普通用户提交的题库进入待审核状态。
2. 管理员审核通过后，其他用户才可见。
3. 驳回或待审核题库只对上传者和管理员可见。
4. 云端题库可缓存到本地，断网后继续使用。

不要反复上传完全相同的文件，系统会使用内容哈希检查重复题库。

## 3. 题库详情

题库详情页包括题型数量、练习统计、学习状态、成绩趋势、题目搜索和分页列表。

可以从这里开始全部练习、按题型练习、随机抽题或需复习练习。“已掌握 / 需复习”按每道题最新一次有效记录计算，不是简单累加历史点击次数。

## 4. 刷题与背题

### 客观题

- 单选题选择一个选项。
- 多选题选择全部认为正确的选项后提交。
- 判断题选择“正确”或“错误”。
- 填空题按空位填写答案。

提交后会显示正确状态、标准答案和解析。答错的题会进入需复习队列。

### 简答题与背记题

简答题默认采用背题策略。查看答案后：

- “已掌握”记录为掌握并进入下一题。
- “再看一遍”记录为需复习并进入下一题。

这两个操作不仅用于翻页，也会改变题目的学习状态。

### 快捷键

| 操作 | 快捷键 |
| --- | --- |
| 上一题 / 下一题 | `←` / `→` |
| 选择选项 | 数字键、A-E、小键盘数字 |
| 提交 | `Enter` |
| 下一题 | `Space` |

小键盘数字兼容 NumLock 开启和关闭状态。输入框正在编辑时，页面不会抢占普通输入。

### 题号面板

点击右下角题号按钮打开题目列表。当前题、已答题和云端已有进度会显示不同状态，点击题号可直接跳转。弹窗打开后背景页面会锁定，系统返回键会优先关闭弹窗。

## 5. 已掌握与需复习

以下行为会影响学习状态：

- 客观题答对：更新为已掌握。
- 客观题答错：更新为需复习。
- 背题点击“已掌握”：更新为已掌握。
- 背题点击“再看一遍”：更新为需复习。

可从首页题库卡片、题库详情、首页“错题练习”或题库详情“开始复习”查看和进入复习。同一道题多次作答时，以最新记录决定当前状态。

## 6. 拍照搜题

点击相机图标打开拍照搜题。首次使用时浏览器需要加载 OCR 模型，等待时间取决于网络和设备性能。

- 让题干尽量铺满画面。
- 保持清晰、平整并减少反光。
- 识别后先检查文字。
- 有错字或漏字时手动修改再搜索。
- 搜索范围是当前设备已有或已缓存的题库。

OCR 识别失败不会损坏题库和练习记录。

## 7. 统计

统计页只显示当前登录账号在当前设备上的本地练习数据，包括总刷题数、正确率、练习次数和最近练习。

云端题库的题目进度会同步，但不同设备上的本地会话列表不会简单合并为一份历史记录。

## 8. 站内公告

管理员可发布公告并设置置顶、发布时间和过期时间。

- 首次进入首页时，未读公告会提醒一次。
- 点击“知道了”后会标记为已读。
- 所有仍有效的公告可在“我的 → 站内公告”中查看。
- 公告弹窗打开时背景不可滚动。

## 9. 意见反馈

进入“我的 → 意见反馈”，选择反馈类型，填写标题和详细内容后提交。管理员回复后可在原反馈中查看处理结果。

待回复且尚无管理员回复的反馈可以自行删除。已回复或已关闭反馈由管理员处理。

## 10. 离线与同步

- 已缓存的云端题库和本地题库可离线练习。
- 离线答题先写入 IndexedDB。
- 网络恢复后，待同步进度会自动回写云端。
- 离开页面时会使用 beacon 做一次兜底提交。
- 同一设备切换账号时，题库、会话、答题记录和统计按账号隔离。

“清除本地数据”会删除当前浏览器里的题库缓存和练习记录，不会删除云端题库或云端进度。

## 11. 安装与更新

浏览器支持时可将应用安装到桌面或手机主屏幕。

- 普通页面检测到安全更新条件时可自动切换。
- 刷题、后台、正在编辑表单或打开弹窗时不会强制刷新。
- 页面出现更新提示时，点击刷新即可切换到新构建。
- 如果仍看到旧界面，关闭 PWA 后重新打开，或在浏览器执行强制刷新。

更新前重要进度会先保存在本地或云端。

## 12. 返回与页面滚动

- 移动端底部“题库、统计、我的、管理”是独立栏目。
- 栏目内进入题库详情、刷题或后台子页面后，返回会逐层进行。
- 跨栏目切换不会形成循环返回。
- 弹窗打开时系统返回键优先关闭弹窗。
- 各主栏目保存各自的滚动位置。

## 13. 常见问题

### 题库总数有数据，但“我的题库”为空

云端题库可能还没有缓存到本地，或当前账号没有本地题库。继续查看云端题库卡片，或点击缓存。

### 需复习数量与历史错题次数不同

需复习按每题最新状态统计。一道题答错多次只计算一题，之后答对或标记掌握会移出需复习队列。

### 进入题库详情或拍照搜题较慢

题库详情需要同步云端题目和进度；拍照搜题首次需要加载 OCR。之后浏览器缓存命中通常会更快。

### 更新后仍是旧界面

先使用页面更新提示。仍未更新时关闭全部应用页面后重新打开，或清理该站点的 Service Worker/缓存后再访问。

### 无法删除反馈

用户只能删除待回复且没有管理员回复的反馈。已回复或已关闭反馈由管理员处理。

### 深色模式显示异常

先刷新到最新版本。如果只有某个页面异常，请通过意见反馈提交页面名称、设备和截图。
